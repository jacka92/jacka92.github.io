Proventor is built on the MEAN stack: MongoDB, Express.js, Angular.js, and Node.js. GPS statistics of various physiological measures are gathered, compiled and housed in the MongoDB database. These measures are visualised alongside injury incidences in order to allow coaches and professional staff to assess the relationships and correlations between these measures and incidences of injury. The application uses a Node.js server, with the native Node driver handling requests to the MongoDB database. An Angular.js SPA has been developed for the front end, with highcharts.js rendering statistics on graphs.

Some preliminary Machine learning work was also applied to test data sets. This was achieved through use of the scikit-learn Python library. Supervised classification algorithms were examined, for instance, Gaussian Naive-Bayes, Support Vector Machines, Decision Trees and K-Nearest-Neighbour.
